# Romarinho AR test package

This package contains nine real-scale GLB models:
- 3 sizes: 21x30, 30x40, 40x50 cm
- 3 frame colors: black, white, light wood

Open `index.html` from an HTTPS web host. The page uses `<model-viewer>` with wall placement and fixed scale. On supported Android devices it opens Scene Viewer/WebXR. On iPhone, model-viewer can generate the USDZ representation when the AR button is tapped, so separate USDZ files are not required for this test.

Important: do not test by opening the HTML directly as a local file. Upload the folder to GitHub Pages or any HTTPS hosting.
